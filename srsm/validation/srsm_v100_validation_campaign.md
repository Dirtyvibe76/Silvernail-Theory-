# SRSM Validation Campaign

The remaining gates cannot be closed by declaration. They require the following executable program.

## RC2 — exact canonical implementation

- implement the local DA2 action;
- generate canonical momenta;
- classify primary and secondary constraints;
- compute the complete Dirac matrix;
- reconstruct all first-class brackets;
- test rank across generic backgrounds.

## RC3 — nonlinear evolution

- use a variational or symplectic constrained integrator;
- evolve solved full-constraint initial data;
- prohibit post-step projection in the pass test;
- measure constraint growth versus resolution and time step.

## RC4 — refinement

- run multiple independently generated lattices;
- compare orientation, topology, and refinement path;
- require common continuum observables.

## RC5 — phenomenology

Use one frozen parameter set to test:

- Newtonian potential;
- PPN coefficients;
- tensor-wave speed and damping;
- homogeneous cosmology;
- at least one compact-object or strong-curvature benchmark.

## RC6 — information-sector tests

- derive the open-system master equation from the CTP action;
- calibrate the decoherence-modulation coefficient;
- test causal-information-current conservation;
- publish negative controls and falsification thresholds.

## Release rule

The label “validated V1.0” is permitted only when every V-gate in the matrix passes from the same executable implementation.
