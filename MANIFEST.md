## Evidence-status rule

A file's presence in this repository does not by itself establish scientific validation. Each recovered artifact must eventually carry one of these labels:

- `PRIMARY_MANUSCRIPT`
- `MATHEMATICAL_SPECIFICATION`
- `EXECUTABLE_SOURCE`
- `RAW_OUTPUT`
- `REGENERATED_OUTPUT`
- `ANALYTICAL_DERIVATION`
- `BOUNDED_PROTOTYPE`
- `ILLUSTRATIVE_DATA`
- `PLANNED_TEST`
- `SUPERSEDED`
- `PROVENANCE_ONLY`

## Completion condition

The repository can be called a full consolidated copy only after:

1. all theory-relevant source files have been inventoried;
2. duplicates have been hash-compared;
3. binary manuscripts and datasets have been added;
4. version history gaps have been recovered or explicitly recorded as missing;
5. every artifact has source and evidence-status metadata;
6. the existing Paper 1 GitHub repository has been imported or referenced without losing commit provenance.
