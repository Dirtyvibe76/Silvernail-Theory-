# Silvernail Theory

This repository is the consolidated archive and working source tree for Aron Silvernail's theoretical research program, including the Silvernail Relational State Model (SRSM), the Silvernail Paradigm, cosmological extensions, numerical prototypes, validation records, and release-gate documentation.

## Repository status

This repository begins as a recovery and consolidation project. The source material was distributed across manuscripts, generated reports, simulation artifacts, and long-form research conversations. Files are preserved according to their evidentiary status rather than being silently treated as one completed executable theory.

The archive distinguishes:

- primary manuscripts and mathematical specifications;
- recovered executable code and raw numerical outputs;
- analytical derivations;
- bounded prototype calculations;
- validation and release-gate reports;
- proposed tests that have not yet been implemented;
- superseded or historical records.

## Scientific status

SRSM Paper 1 studies relational state dynamics, operational geometry, causal propagation, local Hermitian generators, infrared structure, and multidimensional closure in finite models. Later gravity and cosmology work extends the research program but does not yet constitute one fully validated microscopic implementation reproducing General Relativity.

The current V1.0 release gate remains blocked pending a coherent implementation that demonstrates nonlinear first-class closure, projection-free constraint preservation, a stable nonlinear two-mode count, controlled refinement, microscopic derivation of geometric variables and couplings, universal matter coupling, and phenomenological recovery tests.
